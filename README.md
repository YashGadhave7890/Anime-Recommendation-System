# 🎌 Anime Recommendation System

A modern Anime Recommendation Web App built using Machine Learning and Streamlit.

## 🚀 Features
- 🔍 Search-based anime input
- 🎯 Content-based recommendation (TF-IDF + Cosine Similarity)
- 🎴 Anime posters using Jikan API
- 🎨 Modern UI inspired by OTT platforms
- ⭐ Ratings display
- 🔥 Filter options (TV, Movie, OVA)

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
